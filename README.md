# 🚖 Ola Ride Booking Analysis Dashboard | Power BI

## 📌 Project Overview

This project is an interactive **Ola Ride Booking Analysis Dashboard** developed using **Power BI** to analyze ride bookings, revenue performance, vehicle utilization, cancellations, and customer satisfaction metrics.

The dashboard provides a comprehensive view of ride operations and business performance through interactive reports, KPIs, charts, and drill-down analysis. It helps identify booking trends, revenue patterns, cancellation reasons, and vehicle-wise performance to support data-driven decision-making.

---

# 🎯 Objectives

* Analyze overall ride booking performance.
* Track revenue generated through different payment methods.
* Compare performance across vehicle categories.
* Understand customer and driver cancellation behavior.
* Evaluate customer and driver ratings.
* Monitor ride volume and distance trends over time.
